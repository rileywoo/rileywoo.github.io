BearMaps

Language(s): Java
BearMaps was the final project in CS 61B (Data Structures and Algorithms). In the project, I parsed XML from the OpenMapProject and created a rough version of Google Maps. With the A* algorithm, one can navigate from any place in Berkeley to another. Users also can zoom in to yield greater detail in the map, or vice versa. To do this, I implemented the quadtree data structure. There also is a search bar to find locations, implemented with a trie.

I will be deploying the project to Heroku soon so you can see it in action.

Text Editor

Language(s): Java
Coding a text editor was the semi-final project in CS 61B (Data Structures and Algorithms). The editor lets users resize the window, increase/decrease font size, undo/redo actions, file I/O, among other things. I used the JavaFX library for the GUI and various data structures to support efficient runtimes. These structures include a modified doubly-linked list, a modified stack, and ArrayLists.

Rubik's Cube Solver

Language(s): Python
In this project, I created a program that given any Rubik's Cube scramble, will print out a solution. In addition, it returns a TinyURL that leads to a visualization of the cube at each step, so users can follow along. The program helps people who are learning to solve the cube blindfolded, as the solution printed is in terms of the Old Pochmann method, a beginner's method to solving the cube blindfolded. I used recursion and conditional flow. See this GitHub link on how to use it, though I will eventually move it to a browser.
